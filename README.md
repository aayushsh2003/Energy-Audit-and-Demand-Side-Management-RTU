# EnergyMaster: Energy Audit & Demand Side Management Study App

A comprehensive study application for Energy Audit and Demand Side Management (EE 6th Semester), covering 9 detailed units with theory, numericals, and viva preparations.

## 🚀 Features

- **Structured Learning**: 9 comprehensive units extracted from core curriculum data, organized for quick learning and revision.
- **Exam Focused**: Strategic highlighting of "Must Know" topics, common viva questions, and essential numerical formulas.
- **Professional Design**: Clean, technical interface inspired by modern documentation for a distraction-free experience.
- **Interactive Syllabus**: Easy navigation through a collapsible sidebar with progress tracking.
- **Rich Content Components**:
  - **Info Boxes**: categorical tags (Analogy, Topic-wise, etc.) for better context.
  - **Learning Objectives**: Clear goals for every module.
  - **Technical Layouts**: Grids and lists optimized for technical reading.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📂 Project Structure

- `src/App.tsx`: Main application shell with navigation and routing logic.
- `src/components/Sidebar.tsx`: The module directory navigation.
- `src/components/UnitLayout.tsx`: Shared components (Section, InfoBox, List, Grid) for consistent content styling.
- `src/components/units/`: Individual files for all 9 syllabus units.

## 📖 Syllabus Covered

1. **Energy Scenarios**: Conservation laws, audit scope, global and India scenarios.
2. **Types of Energy Audits**: Methodology, financial analysis, and ESCO financing.
3. **Survey Instrumentation**: Electrical, thermal, and speed measurement tools.
4. **Electrical-Load Management**: DSM strategies, VFDs, and power factor.
5. **Energy Audit of Motors**: Classification, efficiency, and conservation measures.
6. **Energy Audit of Lighting Systems**: Lamp types, controls, and efficacy.
7. **Energy Audit Applied to Buildings**: Design strategy and water audit.
8. **Demand Side Management**: Evolution, tariff options, and environment.
9. **Energy Conservation**: Principles, sector-wise conservation, and Legislation (EC Act 2001).

## 💻 Development

To run the app locally:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
